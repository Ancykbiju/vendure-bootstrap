# Vendure Platform Bootstrap

This repository contains bootstrap configurations for the Vendure platform.

## Directory Structure

- `terraform/`: Infrastructure as Code configurations using Terraform
- `scripts/`: Utility scripts for setup and deployment
- `templates/`: Reusable templates for various components
- `.github/workflows/`: GitHub Actions CI/CD workflows

## Getting Started

1. Clone this repository
2. Review and customize the configurations in each directory
3. Run the setup scripts as needed

## Usage

Refer to the README files in each subdirectory for specific instructions.
